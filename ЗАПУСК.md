# Как запустить Barbershop CRM

## Первый запуск
```
cd D:\CRM\barbershop_crm
..\venv\Scripts\python manage.py runserver
```

## Открыть в браузере
http://127.0.0.1:8000

## Логин
- Логин: `admin`
- Пароль: `admin123`

## Мастера (для тестирования)
- Логин: `master_1` / Пароль: `master123`

## Пересоздать тестовые данные
```
..\venv\Scripts\python manage.py seed_data --clear
```
