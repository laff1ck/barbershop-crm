## 1. Назначение и состав сервиса

Сервис **The Cut Barbershop** состоит из двух частей:

| Компонент | Описание |
|-----------|----------|
| **Страница записи** (`/book/`) | Публичная форма для клиентов — доступна без регистрации |
| **CRM-панель** (`/`) | Административная система для сотрудников барбершопа |

Для онлайн-записи регистрация **не требуется**. Достаточно открыть страницу записи и последовательно выполнить четыре шага.

---

## 2. Начало работы

Откройте браузер и перейдите по адресу:

```
http://127.0.0.1:8000/book/
```

**Поддерживаемые браузеры:**

| Браузер | Минимальная версия |
|---------|-------------------|
| Google Chrome | 90+ |
| Mozilla Firefox | 88+ |
| Microsoft Edge | 90+ |
| Safari | 14+ |

## 3. Главный экран и навигация

После загрузки страницы в верхней части отображается **шкала прогресса** из четырёх шагов:

```
① Мастер  ──────  ② Услуга  ──────  ③ Дата и время  ──────  ④ Контакты
```

- Активный шаг выделяется **золотым цветом**
- Кнопка «Далее» становится активной только после выбора на текущем шаге
- Кнопка «Назад» возвращает к предыдущему шагу без потери данных

---

## 4. Шаг 1 — Выбор мастера

Отображается сетка карточек мастеров. Каждая карточка содержит:

| Элемент | Описание |
|---------|----------|
| Фото | Портрет мастера или аватар-заглушка |
| Имя | Отображаемое имя специалиста |
| Специализация | Например: «Барбер», «Стилист» |
| Рейтинг | Звёзды и числовое значение (от 1.0 до 5.0) |

**Как выбрать:** щёлкните по карточке — она выделится золотой рамкой и появится значок ✓. Нажмите «Далее».

---

## 5. Шаг 2 — Выбор услуги

Список услуг выбранного мастера, сгруппированный по категориям:

| Поле | Описание |
|------|----------|
| Название | Наименование услуги |
| Описание | Краткое описание (при наличии) |
| Длительность | Например: «30 мин», «1ч 30мин» |
| Стоимость | Цена в рублях |

Щёлкните по нужной услуге — она выделится золотой полосой слева. Нажмите «Далее».

---

## 6. Шаг 3 — Выбор даты и времени

### 6.1. Выбор даты

В мини-календаре щёлкните по нужной дате.

| Тип дня | Отображение |
|---------|-------------|
| Доступный день | Белый цвет, кликабельный |
| Сегодня | Выделен жирным |
| Прошедшая дата | Серый, недоступен |
| Далее 60 дней | Серый, недоступен |

### 6.2. Выбор слота

После выбора даты справа появляются доступные временные слоты (каждые 30 минут). Уже занятые слоты **не отображаются**.

```
10:00   10:30   11:00   [11:30]   12:00   ...
                         ^^^^^^
                      выбранный слот
```

Нажмите «Далее».

## 7. Шаг 4 — Контактные данные

На экране отображается сводка записи и форма:

| Поле | Обязательное | Описание |
|------|:---:|----------|
| Фамилия | ✅ | |
| Имя | ✅ | |
| Телефон | ✅ | Формат: +7 (999) 123-45-67 |
| Комментарий | ❌ | Пожелания или уточнения |

Нажмите кнопку **«Записаться»**.

---

## 8. Подтверждение записи

После успешной отправки появляется экран с подтверждением:

```
✅ Запись создана!

Мастер:    Алексей Аверьянов
Услуга:    Классическая стрижка
Дата:      15.05.2026
Время:     14:00
Стоимость: 800 ₽
```

Запись сохранена в системе. Для повторной записи нажмите «Записаться ещё раз».

## 9. Часто задаваемые вопросы

**Можно ли записаться без телефона?**  
Нет — телефон используется для идентификации клиента в системе.

**Как отменить запись?**  
Свяжитесь с барбершопом по телефону. Самостоятельная отмена через сайт не предусмотрена.

**Почему нет свободных слотов?**  
Все слоты могут быть заняты, либо мастер не работает в выбранный день. Попробуйте другую дату или другого мастера.

**На каких устройствах работает сервис?**  
Сервис адаптирован для компьютеров, планшетов и смартфонов (минимальная ширина экрана — 320px).
